# RollCall - Frontend

This directory contains the frontend web application for RollCall, built using **Next.js** and **Tailwind CSS**.

## Directory Structure

-   `src/app/`: App Router pages and layouts (e.g., `dashboard`, `login`).
-   `src/components/`: Reusable UI components.
-   `src/utils/`: Utility functions and API helpers.
-   `public/`: Static assets.

## Setup & Running

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```
    Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Key Features

-   **Dashboard**: Overview of attendance status.
-   **Authentication**: Login and registration pages.
-   **Responsive Design**: Built with Tailwind CSS for mobile and desktop support.
